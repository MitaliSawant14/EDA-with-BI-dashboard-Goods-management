# Inventory and Product Analysis

## Project Overview

This project focuses on analyzing inventory and product data to uncover insights related to sales performance, wastage, profit, and product categories. The analysis was performed using Python for Exploratory Data Analysis (EDA), SQL for data storage and querying, and Power BI for interactive dashboard visualization.

The goal of this project is to help businesses understand product performance, optimize inventory management, and identify opportunities to reduce wastage while improving profitability.

## Problem Statement

Businesses often struggle to monitor inventory efficiency and product performance. High wastage, uneven sales distribution, and category-level profit differences can affect overall profitability. This project aims to analyze inventory data to identify trends, patterns, and insights that support better business decisions.

## Dataset Description

The dataset contains transaction records related to inventory and products across multiple categories. It includes information such as product name, category, region, sales, wastage, profit, and other operational metrics.

## Tools and Technologies Used

* Python (Pandas, NumPy, Matplotlib, Seaborn, Plotly)
* SQL (MySQL)
* Power BI
* Google Colab

## Exploratory Data Analysis (EDA)

The following analysis was performed:

* Data cleaning and preprocessing
* Handling missing values and incorrect data types
* Univariate analysis to understand distribution of variables
* Bivariate analysis to identify relationships between variables
* Detection of outliers
* Analysis of product categories, sales distribution, and wastage patterns

## Key Insights

* Certain product categories generate higher revenue but also show high wastage.
* Sales are unevenly distributed across categories and regions.
* Some products contribute significantly to total revenue, indicating a Pareto pattern.
* Categories with high wastage may require improved inventory control.

## Power BI Dashboard

An interactive Power BI dashboard was created to visualize:

* Revenue distribution
* Category-wise performance
* Wastage trends
* Product-level contribution to total revenue
